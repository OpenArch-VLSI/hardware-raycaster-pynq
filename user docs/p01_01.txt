# 01 Architecture Overview

## What the design does

This is a hardware raycaster in SystemVerilog, inspired by Wolfenstein 3D. It casts 640 rays per frame through a 2D map grid, computes wall distances, and renders textured walls at 640x480 with 24-bit color over DVI (TMDS). Player position and camera direction are updated via 6 key inputs between frames. Double-buffered frame storage prevents screen tearing. The design is FSM-based with aggressive resource sharing: the raycaster is iterative, not pipelined.

Target platform: Gowin Tang Primer 20K (GW1N-1), yosys-slang + nextpnr + Apicula.
